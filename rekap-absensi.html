Panduan Modul

Cara pakai Rekap Absensi

Gunakan halaman ini untuk melihat laporan kehadiran berdasarkan periode tanggal dan akses masing-masing pengguna.

1

Pilih periode tanggal

Pilih tanggal awal dan tanggal akhir untuk menentukan periode laporan yang ingin dilihat.

2

Pilih karyawan sesuai akses

Pimpinan dan supervisor dapat melihat data seluruh karyawan. Karyawan hanya dapat melihat data miliknya sendiri.

3

Gunakan filter tambahan

Gunakan filter outlet dan status untuk mempersempit hasil laporan, misalnya hadir, terlambat, pulang cepat, atau belum check out.

4

Refresh dan export

Tekan Refresh untuk memperbarui tampilan laporan. Gunakan Export PDF untuk menyimpan laporan per karyawan ke Drive. Jika data belum ada, sistem tetap membuat template PDF kosong. Export CSV digunakan jika laporan perlu diunduh.

Catatan operasional

Gunakan data rekap sebagai bahan pengecekan kehadiran. Jika ada data yang belum sesuai, hubungi supervisor atau admin untuk pemeriksaan lebih lanjut.
